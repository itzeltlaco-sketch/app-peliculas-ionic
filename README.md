# Aplicación de Películas (Ionic)

## Descripción general
Aplicación desarrollada con Ionic y Angular que permite visualizar una lista de películas, ver detalles y realizar búsquedas.  
Proyecto elaborado para la materia de *Aplicaciones Móviles*.

## Tecnologías utilizadas
- Ionic Framework  
- Angular  
- TypeScript  
- HTML / CSS  
- API de películas (o datos locales)
